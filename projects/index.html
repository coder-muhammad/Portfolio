ICC Champions Trophy 2025 Project

The ICC Champions Trophy 2025 website is a comprehensive platform created to provide detailed information about the upcoming cricket tournament. Built using only HTML, this project showcases my ability to structure web content effectively and deliver essential event information.

Purpose: The main goal of this project is to offer a user-friendly website where cricket enthusiasts can easily access information about the ICC Champions Trophy 2025, including match schedules, team profiles, and venue details.

Steps Taken to Create the Project

  1. Planning the website structure and content layout.
  2. Creating HTML pages for different sections, such as schedule, teams, and venues.
  3. Organizing and linking pages for smooth navigation.
  4. Ensuring accessibility and user-friendly design.
  5. Testing the website for functionality and user experience.

Key Pages

Homepage (index.html)

Provides a brief introduction to the ICC Champions Trophy 2025, highlighting key dates and events. Includes links to various sections of the site, such as schedules, teams, and venues.

About Page (about.html)

Offers detailed information about the history and significance of the ICC Champions Trophy.

Schedule Page (schedule.html)

Displays a detailed schedule of all matches, including dates, times, and locations. Allows users to filter matches by team or venue for easy navigation.

Teams Page (teams.html)

Contains profiles for each participating team, showcasing team history, player rosters, and key statistics.

Contact Page (contact.html)

Contains a form for visitors to get in touch with the tournament organizers. Provides contact information and social media links.

Challenges and Solutions

Content Organization: Ensuring that all information is presented clearly and logically required careful planning and structuring of HTML elements.

User Experience: Creating an intuitive and user-friendly navigation system was crucial to keep visitors engaged and make it easy for them to find information.

Accessibility: By using semantic HTML, I ensured that the website is accessible to all users, including those using screen readers.
